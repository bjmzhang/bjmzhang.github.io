## CSS全局声明*与body的区别
- body有继承性
```html
举个例子就明白了：
<h1>标签默认的字体大小为"font-size: 2em;"，如果用:
* {
    font-size:20px;
}
声明全局样式，则<h1>标签的字体大小会更改为"font-size: 20px;"；而如果用:
body {
    font-size:20px;
}
声明全局样式，则<h1>标签的字体大小仍为"font-size:2em;"，只是计算由原来浏览器的默认字体大小16px × 2 = 32px更改为20px × 2 = 40px；
```
